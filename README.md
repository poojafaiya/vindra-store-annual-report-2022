# vindra-store-annual-report-2022
This project presents a comprehensive annual sales analysis for Vrinda Store for the year 2022. The objective of this report is to identify key business insights and performance trends using data analytics techniques.
# vinra store-daya- analysis(dashboard creating MS EXCEL)

## project objective
the vindra store wants create an annual sales report for 2022. that's why the owner of the vindra 
store can understand thair coustomers and manage sales in 2023 and 2024.
## data used
-<a href ="https://github.com/poojafaiya/vindra-store-annual-report-2022/blob/main/Vrinda%20Store%20Data%20Analysis%20(1).xlsx">dataset</a>


## qustions(kpis)

- What was the total sales trend month-wise for the year 2022?
- How many orders were placed each month and what were the highest and lowest months?
- What percentage of sales came from women vs men customers?
- What was the overall order fulfillment rate (delivered vs returned/cancelled/refunded)?
- Which states generated the highest revenue and what are the top 5 states?
- Which customer age group placed the maximum orders (Teen, Adult, Senior)?
- How did order distribution vary across age and gender segments?
- Which sales channel performed the best (Amazon, Flipkart, Myntra, etc.)?
- What percentage contribution did each sales channel have in total revenue?
- What product categories contributed most to sales? (based on slicer filter)

## process
- Verified and cleaned raw data by handling missing values, duplicates, and formatting issues.
- Ensured data consistency by correcting data types, standardizing text fields, and formatting date columns.
- Prepared the dataset and created calculated columns for month, state, age group, gender, and channel analysis.
- Created pivot tables based on business questions (sales trends, gender insights, top states, order channels).
- Designed an interactive dashboard by combining pivot charts, pivot tables, and KPIs into a single Excel sheet.
- Applied slicers for category, channel, and month to make the dashboard dynamic and user-frien
